# Colibri Detalles

Sitio web profesional para mostrar trabajos, servicios y recibir cotizaciones.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Editar contenido simple

- Datos principales: `content/site.json`
- Trabajos: `content/trabajos.json`
- Servicios: `content/servicios.json`

Para agregar fotos propias, colocalas en `public/images/trabajos/` y usa rutas como:

```txt
/images/trabajos/mi-foto.jpg
```

Mientras no tengas fotos cargadas, el sitio usa imagenes externas de ejemplo.
